#### Description
The Patient Story Card contains an image, title, and a link with a label. All fields are required.

This component is used in the multiple patient story banner. 

The image must be a square ratio.

#### Variables
~~~
image:
  alt: text / required
  src: media / required
title: text / required
link:
  url: href / required
  label: text / required
~~~
