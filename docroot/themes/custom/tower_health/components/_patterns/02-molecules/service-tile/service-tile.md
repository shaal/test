#### Description
The Service Tile is a teaser for a core service. It lists the core service with a link to the core service's page as well as a set of sub-services for that core service.

#### Variables
~~~
service:
  name: short text / required
  url: href / required
  subservices:
    name: short text/ required
    url: href / required
~~~
